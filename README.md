Journaling app

Created with React, Redux and Rails   
MVP done but this project is still in development     

CD in the backend   
Run ```bundle install```   
Run ```rails s```   
CD in the frontend   
Run ```npm install```    
Run ```npm start```   

A user can add posts, edit and delete them    
The posts are searchable by their tags
